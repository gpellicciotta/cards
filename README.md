# Digital Cards

Simply digital cards whose URL can be shared.

Some cards can be customized via URL parameters: use ?help to see what is possible (if anything).

Following cards are available:
  - [Happy Birthday](https://www.pellicciotta.com/cards/happy-birthday.html): with message and fireworks
  - [Celebrate](https://www.pellicciotta.com/cards/celebrate.html): with message and fireworks
  - [Merry Christmas](https://www.pellicciotta.com/cards/merry-christmas.html): with snowflakes falling
  - [Happy New Year](https://www.pellicciotta.com/cards/happy-new-year.html): with countdown and fireworks
